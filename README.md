## 🚀 Quick Start

**Use it online — no install needed:**

👉 **[lebedevnet.github.io/ReadmeForge](https://lebedevnet.github.io/ReadmeForge/)**

```
1. Fill in your name, role & bio
2. Select your tech stack — click chips to toggle
3. Add your social links
4. Pick a theme & enable widgets
5. Click ✨ Generate README
6. Copy → paste into your GitHub profile repo
```

**Or run it locally:**

```bash
git clone https://github.com/lebedevnet/ReadmeForge.git
cd ReadmeForge
open index.html   # or just double-click the file
```

No npm. No build step. No config. It just works.
