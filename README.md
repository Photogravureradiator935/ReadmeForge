## ✨ What is ReadmeForge?

**ReadmeForge** is a free, browser-based tool that generates a fully customized `README.md` for your GitHub profile. Fill in a simple form — click Generate — copy and paste. Done in under 2 minutes.

> 💡 A great GitHub profile README gets you more followers, job offers, and open source contributions. ReadmeForge makes it effortless.

---

## 🎯 Features

- 🧩 **60+ technologies** — Languages, Frontend, Backend, DevOps, AI/ML with real logos
- 🎨 **16 beautiful themes** — Tokyo Night, Dracula, Nord, Synthwave, Aura and more
- 🌍 **Spoken languages** — Show what languages you speak with a visual level indicator
- 🔗 **10 social links** — LinkedIn, Twitter/X, Telegram, YouTube, Twitch, Dev.to and more
- 📊 **12 widgets** — GitHub Stats, Streak, Trophies, Activity Graph, WakaTime, Spotify and more
- ⌨️ **Animated typing** — Add a typing effect header to your profile
- 📋 **One-click copy** — Your README.md is ready to paste instantly
- 🌐 **Works in browser** — No install, no backend, no sign-up required

## 🚀 Quick Start

**Option 1 — Use it online (recommended):**

👉 **[lebedevnet.github.io/ReadmeForge](https://lebedevnet.github.io/ReadmeForge/)**

**Option 2 — Run locally:**

```bash
git clone https://github.com/lebedevnet/ReadmeForge.git
cd ReadmeForge
open index.html
```

That's it. No npm, no build step, just open the file.
